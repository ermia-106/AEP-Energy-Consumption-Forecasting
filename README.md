# ⚡ AEP Energy Consumption Forecasting

A complete Time Series Forecasting project using the **SARIMA (Seasonal ARIMA)** model to predict hourly electricity consumption for the American Electric Power (AEP) grid.

The project covers the entire forecasting pipeline, including data preprocessing, exploratory data analysis (EDA), stationarity testing, SARIMA model selection, model evaluation, future forecasting, and model serialization.

---

## 📸 Project Overview

This project performs:

- Data preprocessing
- Missing timestamp interpolation
- Duplicate timestamp handling
- Exploratory Data Analysis (EDA)
- Augmented Dickey-Fuller (ADF) Stationarity Test
- ACF & PACF Analysis
- Seasonal Differencing
- Multiple SARIMA Model Comparison
- Model Selection using AIC & BIC
- Model Evaluation
- Forecast Visualization
- Model Serialization
- Future Energy Consumption Forecasting

# 📄 Files Description

### AEP_hourly.csv

Original hourly energy consumption dataset.

---

### AEP-Energy-Consumption-Forecasting.py

Main project script that performs:

- Data preprocessing
- Data cleaning
- Stationarity testing
- ACF & PACF analysis
- Seasonal differencing
- SARIMA model comparison
- Best model selection
- Model training
- Model evaluation
- Forecast visualization
- Final model training
- Model saving
- One-week forecasting

---

### AEP-Forecast.py

Loads the saved SARIMA model and predicts future energy consumption for any number of hours specified by the user.

No retraining is required.

---

### AEP-One-Week-Forecast.csv

Sample forecast generated by the trained model.

---

### AEP-SARIMA-Model.pkl

Serialized trained SARIMA model.

This file allows instant forecasting without rebuilding the model.

# ▶️ Usage

## Train the Model

```bash
python AEP-Energy-Consumption-Forecasting.py
```

This script will:

- preprocess the dataset
- compare SARIMA models
- train the best model
- evaluate performance
- save the trained model
- generate a one-week forecast

---

## Forecast Future Consumption

```bash
python inference.py
```

Example

```text
Enter The Number of Hours for Forecasting: 48
```

The saved model is loaded instantly and forecasts the requested number of hours.
